📊 Log File Analyzer (Python Mini Project)
📌 Project Overview

The Log File Analyzer is a Python mini project that reads a system log file and analyzes different types of log messages such as INFO, WARNING, and ERROR.

The program processes the log file line by line, counts the occurrences of each log level, and extracts error messages to generate a summary report.

This project demonstrates Python file handling, string manipulation, loops, and basic log data analysis.

🚀 Features

Reads a log file

Counts different log levels:

INFO

WARNING

ERROR

Extracts error messages

Displays a summary report

Beginner-friendly Python implementation
